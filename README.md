# FitLife

Программа для фитнес-трекера, которая позволяет расчитывать ИМТ и дневную норму воды для пользователя

## Запуск

1. Клонируйте репозиторий на свой компьютер
2. Запустите командную строку (например Git Bash)
3. Перейдите в директорию проекта
4. Создайте виртуальное окружение:
    `python -m venv venv`
    `source venv/Scripts/activate`
5. Установите зависимости из requirements.txt
    `pip install -r requirements.txt`
6. Запуск программы
    `python fit_life.py`
